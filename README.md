# Google Homepage Clone

A simple clone of the Google Homepage built using web development technologies. This project recreates the design and layout of Google's main search page.

## 🚀 Features
- Google homepage UI design
- Responsive layout
- Search bar design
- Navigation bar
- Clean and simple interface
- Mobile-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (if used)
